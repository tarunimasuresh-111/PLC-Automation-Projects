Automatic Tank Filling System
A PLC ladder logic program that automates the filling of a water tank using a level sensor.
The pump starts when the start button is pressed and continues running through a latching circuit. Once the tank reaches the preset level, the sensor stops the pump automatically.

Software
PicoSoft 6

Concepts Used
Ladder Logic (LAD)
Latching Circuits
Sensor-Based Control
Motor Control
Process Automation
