Forward Reverse Interlocking

A PLC ladder logic program for controlling a motor in forward and reverse directions using electrical interlocking.
The program prevents the forward and reverse outputs from being energized at the same time, ensuring safe motor operation. Latching contacts are used to maintain the selected direction until another command is given.
