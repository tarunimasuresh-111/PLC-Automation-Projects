Traffic Light Control System

A PLC ladder logic program that simulates the operation of a traffic light using timers and sequential control.

The system cycles through red, yellow, and green lights in a predefined sequence. Timer functions control the duration of each signal, allowing the traffic light to operate continuously.

Software - PicoSoft 6

Concepts Used
Ladder Logic (LAD)
Timers
Sequential Control
Multi-Output Control
Traffic Signal Logic
