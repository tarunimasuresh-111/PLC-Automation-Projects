Conveyor Bottle Counting System

A PLC ladder logic program for counting bottles on a conveyor using a sensor and counter function.
The conveyor starts when the start button is pressed. Each bottle detected by the sensor increments the counter. Once the preset count is reached, the conveyor stops and an alarm is activated.

Software - PicoSoft 6

Concepts Used
Ladder Logic (LAD)
Counter Functions
Sensor-Based Counting
Conveyor Control
Alarm Indication
